# Project Requirements Specification

## Source Guidance
This project is informed by publicly available CDC guidance on laboratory
exposure to *Burkholderia pseudomallei* and *B. mallei*.

## Core Features
- Educational exposure risk categorization
- Clear reporting and notification pathways
- Conceptual follow-up timelines
- Explicit handling of *B. mallei* serology limitations

## Explicit Exclusions
- No laboratory techniques
- No organism identification
- No clinical treatment instructions
- No antibiotic dosing
- No operational response protocols

## Ethical Principles
- Synthetic data only
- Education and awareness focus
- Systems-based perspective
